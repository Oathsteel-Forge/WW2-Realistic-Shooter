Controls
WASD to move
Mouse to look
E to interact
Space to jump
Q (hold) to drop equipped item
O toggle god mode (for testing purposes)
P toggle pause menu (acts like a multiplayer game where everything is still going on)
CTRL crouch
1 switch to primary
2 switch to secondary
Left click to shoot
Right click (hold) to ADS
R to reload
